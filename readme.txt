Chào thầy

Nhóm em gửi kèm link google drive cho thầy xem xét, nếu thầy không tiện chạy trên máy tính cá nhân ạ

Nếu thầy chạy bằng file được download từ github thì:

    Đầu tiên: Vào thư mục dự án đó mở cmd/terminal chạy lệnh my_env\Scripts\activate.bat
                để mở lên môi trường ảo chúng em đã tạo để đồng bộ version các thư viện

    Tiếp theo: Quan sát cmd/terminal nếu thấy được đường dẫn dạng như:
                (my_env) C:\Users\hovan\OneDrive\Máy tính\New folder\NCKHDL_FWIs>
                
    Cuối cùng các file python bình thường
